# GameOfLife
Conway's Game of Life in C#

The Project was created in a C# Visual Programming class that I took at my University. I've added a the following GUI features:
- Play/Pause
- Fast/Slow
- Upload shapes
- Pre-built shapes
- Mirror x and y axis
- Resize grid
- Change color
